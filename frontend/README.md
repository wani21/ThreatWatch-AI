# SentinelAI React Frontend

This directory is reserved for the SentinelAI React + TypeScript frontend application.

## Structure Skeleton
The folder structure is organized as follows:
- `src/pages/`: Page components for routing (e.g. Dashboard, Login, Alerts, Users).
- `src/components/`: Reusable UI components (buttons, charts, metric cards, tables).
- `src/services/`: Direct API communication layers connecting with our FastAPI backend.
- `src/hooks/`: Custom React hooks for security analytics state or authentication logic.
- `src/layouts/`: Global page layout templates (Dashboard sidebar layouts, Auth layouts).
